# MoodChef (ムードシェフ) 🍳✨

今の気分にぴったりのレシピを提案する、気分ベースのレシピ提案アプリです。

## 🌐 アプリURL
[https://t225t091-wq.github.io/my-app-2026/](https://t225t091-wq.github.io/my-app-2026/)

## ✨ 特徴
- **直感的な操作:** 6つの気分（元気いっぱい、のんびり、冒険したいなど）から選ぶだけ。
- **魔法の提案:** 「考え中...」の演出のあと、あなたに最適なレシピが1つ表示されます。
- **美しく見やすいレイアウト:** 材料と手順を清潔感のあるデザインで表示。モバイルでも快適です。
- **画像不要の軽量設計:** CSSを駆使したグラフィカルなデザイン。

## 📖 使い方
1. **気分を選択:** トップ画面に表示されている「今の気分」に一番近いカードをクリックします。
2. **レシピを確認:** 提案された料理の材料と作り方を確認します。
3. **選び直す:** 別の気分で探したい場合は「選び直す」ボタンをクリックして戻ることができます。

## 🛠 カスタマイズ方法（開発者向け）
新しいレシピを追加したり、既存の内容を変更したりするには、`data.js` ファイルを編集します。

```javascript
// data.js の recipes 配列にオブジェクトを追加
{
    id: 7,
    title: "新しい料理名",
    mood: "対応する気分",
    emoji: "アイコン",
    time: "調理時間",
    difficulty: "難易度",
    ingredients: ["材料1", "材料2"],
    instructions: ["手順1", "手順2"]
}
```

## 🚀 デプロイ
このプロジェクトは HTML/CSS/JS の静的ファイルのみで構成されているため、GitHub Pages で簡単にホストできます。
